## EasyWaterfall
1. Recopilación de Datos
EasyWaterfall no recopila, almacena ni comparte ninguna información personal o datos empresariales identificables. La visualización procesa los datos proporcionados por el usuario dentro del entorno seguro de Power BI únicamente para fines de renderizado gráfico.

### 2. Transmisión de Datos
Esta visualización no transmite datos fuera de la red de su organización ni a ningún servicio de terceros. Toda la lógica de procesamiento se ejecuta localmente en el navegador del usuario o en el cliente de Power BI Desktop/Mobile.

### 3. Uso de Servicios de Microsoft
Al ser una visualización personalizada para Power BI, EasyWaterfall utiliza las APIs estándar de Microsoft Power BI. Cualquier dato estadístico o de telemetría técnica (como errores de carga o rendimiento) es gestionado directamente por Microsoft de acuerdo con sus propias políticas de privacidad y cumplimiento.

### 4. Almacenamiento
No utilizamos cookies, almacenamiento local (LocalStorage) ni bases de datos externas para rastrear el comportamiento del usuario o guardar datos del informe.
